# PAP
Este é o repositório da minha PAP (Prova de Aptidão Profissional). Na altura, decidi desenvolver uma loja online de tecnologia, mas queria ir um bocado mais longe do que apenas fazer a montra para o cliente.
Por isso, construí também uma aplicação de gestão de stock integrada para controlar tudo o que se passava no armazém. O projeto inteiro foi montado a usar OutSystems, que foi a ferramenta com que aprendi a trabalhar durante o meu estágio. Foi o meu primeiro grande projeto a ligar a parte de vendas com as necessidades reais de um back-office.
